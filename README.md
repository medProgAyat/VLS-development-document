# ابزار تولید سند الزامات رسمی

یک سیستم حرفه‌ای برای تولید و مدیریت اسناد الزامات تیم‌های نرم‌افزاری و مهندسی.

## ویژگی‌ها

- ✅ طراحی رسمی و حرفه‌ای مناسب برای پرینت
- ✅ مدیریت الزامات با قابلیت افزودن، ویرایش و حذف
- ✅ تفکیک الزامات به دو دسته "نیاز" و "خواسته"
- ✅ تعیین میزان اهمیت برای خواسته‌ها
- ✅ شماره‌گذاری خودکار الزامات
- ✅ ذخیره‌سازی خودکار تاریخ آخرین تغییر
- ✅ قابلیت انتخاب تاریخ با تقویم
- ✅ ذخیره‌سازی داده‌ها در پایگاه داده NoSQL محلی
- ✅ بهینه‌شده برای پرینت با حذف عناصر غیرضروری
- ✅ واکنش‌گرا و سازگار با دستگاه‌های مختلف
